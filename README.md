## Getting Started

#### Installation

1. Clone the Repo

```
git clone https://github.com/mnxmnz/Twitter-Clone-Frontend.git
```

```
cd Twitter-Clone-Frontend
```

2. Install Project Packages

```
npm install
```

3. Run the Project (Dev Mode)

```
npm run dev
```
